p1
안녕하세요 이번에 발표를 맡게된 42팀의 팽건우 입니다.
저희는 온라인 가상 안경 피팅 룸이라는 주제를 맡게 되었습니다.
이 설명에 앞서 저희 팀을 소개 하겠습니다.

p2
저 기계설계공학부, it정보공학부를 다니고 있는팽건우와
it정보공학부 유나현으로 구성되어 있는 팀 입니다.
밑에는 각자의 github 주소임으로 많이 들러주시기 바랍니다.

p3
가상 안경 피팅 룸을 간략하게 설명을 하면
가상환경에서 안경을 써 볼 수 있는 서비스라고 볼 수 있습니다.

p4 
이러한 서비스를 생각하기 전에 많은 문제점이 있었습니다.
크게 3가지로 나눌 수 있습니다.
확인, 선택, 전염병 & 손상입니다.

확인은 안경점에 갔을 경우 안경 상표,렌즈 없는 안경을 써서
나의 모습을 잘 확인 할 수 없다는 점입니다. 이로 인해 선택에 대해 고민을 많이 하게 됩니다.

선택은 확인에서 확장된 문제로 나의 모습을 잘 확인 할 수 없고 같이 있지 않은 경우 
혼자서만 모든 것을 선택해야 하기 때문에 구매를 한 후에 후회를 할 수 있는 문제점이 있습니다.

마지막으로 전염병과 손상은 여러 사람이 안경을 써보고 그냥 넣어 놓을 때가 많습니다.
즉 많은 사람의 손을 탄다는 의미가 됩니다. 이로인해 전염병의 위험에 노출이 되며
안경이 손상되기 쉽습니다.  

p5
그래서 다음과 같이
이러한 문제를 해결하기 위해 대부분의 회사에서 기술을 냈습니다.
그러나 이런 기술의 현황과 한계를 보시면 다음과 같습니다.
읽어준다.
그래서 앞선 문제점들을 해결하기 위해 저희 가상 안경 피팅룸 만의 특징을 보면 다음과 같습니다.

p6
Live 실시간이므로 로딩 속도를 보안하고 안경을 선택하여 
자신의 얼굴에 안경을 피팅하여 전염성 & 손상 문제를 막습니다.
동영상으로 저장을 하여 확인 ,선택 관련 문제를 해결하는 것입니다.

p7
저희는 visual studio code 와 python에 있는 library OpenCV를 이용해서
개발을 할 예정입니다.

p8
조금 더 자세히 들어가 보면
세부 일정을 크게 3가지로 나눴습니다.
얼굴인식, 안경 합성, 동영상 
얼굴인식은 캐스 케이드 분류기를 사용해서 얼굴을 인식할 예정입니다.
기간은 약 3주 정도 잡았습니다.

안경 합성은 눈 위치에 맞게 PNG 파일을 합성할 예정입니다.
좌표등을 찾아서 맞춰야 하기 때문에 3주 정도 잡았습니다.

마지막으로 동영상 저장은 OpenCV 기능을 사용하고 
어떻게 처리할 지 생각을 해봐야해서 1주 정도 잡았습니다.

추가로 시간이 더 된다면 웹이나 앱에 적용까지 해보려고 합니다.

또한 저희 프로젝트는 밑에 나와 있는 github 주소에서 협업을 하고 있습니다.
